Detailed Description

Klasa reprezentująca sensor robota micromouse.

Klasa reprezentuje sensor myszy. Sensor posiada ustalony kierunek w którym bada obecność ściany. Na podstawie pozycji myszy ustala czy w danym punkcie znajduje się ściana czy puste pole.
