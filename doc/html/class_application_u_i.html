Detailed Description

Klasa odpowiadająca za interfejs użytkownika.

Klasa odpowiada za wyświetlanie interfejsu użytkownika, w szczególności menu wyboru pliku labiryntu, samego labiryntu oraz wyświetlanie pozycji myszy i jej ścieżki.
